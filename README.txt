# 2D-World-Generator

This project was developed in the context of the lecture GameEngines.
It was the final project in this lecture.

It generates 2D-World-Maps.
These maps contain different bioms (Mountain, Woods, Meadows) and different Assets ( MassiveRock, Walls, Trees, Bushes, Grass, Stones and Water).
It works with Perlin Noise and Open Simplex Noise.
